# Codex/OMX Rules & Path Standards Compatibility

This directory records how Claude-native CCGS rule files and scoped path standards map into Codex/OMX-compatible enforcement surfaces.

Source-of-truth remains:

- `.claude/rules/*`
- root `CLAUDE.md`
- scoped `CLAUDE.md` files in repo subtrees

Codex/OMX compatibility surfaces include:

- `AGENTS.md`
- required-reading directives for scoped `CLAUDE.md`
- hook compatibility entrypoints under `.codex/hooks/`
- workflow/agent compatibility docs under `.codex/skills/` and `.codex/agents/`
- the mapping documents in this directory
